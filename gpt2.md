## 🧭 1. Създаване на Watchlist в TradingView

### Стъпки:

1. Влез в **TradingView**
2. В лявото меню → **Watchlist** (иконата със списък)
3. Натисни **+ Add Watchlist**
4. Дай име, напр.
   👉 **“MINING – Gold / Silver / Copper”**

## 🔔 3. Настройване на Alerts (най-важната част)

### 🔹 A. Alert при ключово ниво (Breakout / Breakdown)

1. Отвори графиката (напр. **AYA**)
2. Избери таймфрейм (4H или 1D)
3. Кликни **дясно върху цената** → **Add Alert**
4. Настройки:

   * **Condition:** Crossing / Crossing Up / Crossing Down
   * **Value:** напр. `15.00`
   * **Timeframe:** 4H или 1D
   * **Expiration:** No expiration
   * ✔️ Notify on app / email

📌 Пример за AYA:

* 🔔 Alert при **15.00** → възможен бичи сигнал
* 🔔 Alert при **12.80** → риск / стоп зона

---

### 🔹 B. Alert при индикатор (по-напреднало)

#### RSI (силно полезно)

1. Добави **RSI (14)**
2. Дясно върху RSI → **Add Alert**
3. Примери:

   * RSI **Crossing Up 30** → възможен отскок
   * RSI **Crossing Down 70** → потенциална корекция

#### Moving Averages

* Alert при **цена над 50 EMA / 200 EMA**
  → често началото на нов тренд

---

### 🔹 C. Alert при обем (Volume spike)

1. Добави **Volume**
2. Add Alert → **Volume > X**
3. Полезно за:

   * пробиви
   * новини за находища
   * институционален интерес

---

## 🧠 4. Мини стратегия с Alerts (много ефективна)

### Бичи сигнал (пример):

✅ Цена > 15.00
✅ RSI > 50
✅ Обем ↑

👉 тогава: **следиш за вход / добавяне**

### Мечи сигнал:

❌ Цена < 12.80
❌ RSI < 40
👉 защита / излизане
